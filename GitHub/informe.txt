#imprimir Hola Mundo
Repositorio creado para archivo basico con codigo "Hola mundo"
Pasos para crearlo y subir a GitHub

#Pasos
1. Creacion de un archivo local en el escritorio.
2. Nombramos la carpeta y la abrimos con VS Code.
3. Creamos un respositorio en GitHub para enlazar a nuestro archivo.
4. Abrir Terminal y comenzar a hacer cambios y configuraciones.
5. Subimos el archivo al repositorio remoto en GitHub.

#Comentario extra
Nos ayudamos con guia proporcionada por el Ing. Luis Jaramillo, asi mismo con videos de youtube para mejor entendimiento.
